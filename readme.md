## apes-planet-box

### 前端快速开始

```javascript
git clone git@github.com:ohray666/apes-planet-box.git
cd web
cnpm i  // 依赖包安装
npm run build:dll   // dllplugin进行打包
npm run build:dev // 开发环境打包
npm start // 启动项目
```
